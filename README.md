# Todo List móvil

Proyecto base para una app móvil de tareas con React Native + Expo.

## Lo que ya quedó listo
- Pantalla principal con lista de tareas
- Crear tareas
- Marcar tareas como completadas
- Eliminar tareas
- Filtro: todas, pendientes y completadas
- Guardado local con AsyncStorage

## Antes de ejecutar
En este equipo falta instalar Node.js.

### 1. Instala Node.js LTS
Descarga e instala la versión LTS desde:
https://nodejs.org/

Luego cierra y vuelve a abrir VS Code.

### 2. Instala las dependencias
Abre una terminal en esta carpeta y ejecuta:

npm install

### 3. Inicia la app

npx expo start

## Cómo probarla en tu celular
1. Instala **Expo Go** en Android o iPhone.
2. Ejecuta `npx expo start`.
3. Escanea el QR desde Expo Go.

## Archivos principales
- `App.js`: interfaz y lógica de la app
- `package.json`: scripts y dependencias
- `app.json`: configuración de Expo
